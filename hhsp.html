New HHSP Website

The Challenge

I was asked to completely redesign and code the new Hudson and Holland Scholars Program (HHSP) website. This is the external facing website that students and parents would go to learn more about the program and stay up to date on news.

The Solution

I created a new responsive website for the scholars program. It was coded with HTML5, CSS3, JavaScript, and jQuery. I used the new branding styles of Indiana University's web presense. The website is located at http://www.iub.edu/~hhsp/

The Process

I started by understanding the problem. I interviewed my director and associate director to uncover what the needs were for this new website. The primary needs were an improved website that was modern and followed the Indiana University brand standards. I then started sketching, created wireframes, and high fidelity mock ups. I single handedly designed and developed this site, with the help of preexisting code from Indiana University's other websites.

Future Goals

I need to add content and optimize the code. After that's finished, I will run some usability testing to make sure the website is usable and understandable. I will continue testing throughout the semester and ensure 100% cross-broswer compatibility and optimal experiences on both the mobile and web version.
